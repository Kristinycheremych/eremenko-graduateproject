3) Маршрут для добавления нового этапа для указанного проекта.
4) Маршрут для обновления данных о конкретном этапе проекта.
5) Маршрут для удаления конкретного этапа проекта.

Набор маршрутов для статуса задач

1) Маршрут для получения списка всех статусов задач для определенного этапа проекта.
2) Маршрут для создания нового статуса задачи для конкретного этапа проекта.
3) Маршрут для удаления статуса задачи по его уникальному идентификатору. 

Набор маршрутов для задачи

1) Маршрут для получения всех задач, относящихся к определенному статусу задачи.
2) Маршрут для создания новой задачи для определенного статуса задачи. 
3) Маршрут для обновления информации о конкретной задаче.
4) Маршрут для удаления задачи по ее уникальному идентификатору.
5) Маршрут для перемещения задачи между статусами задач.
